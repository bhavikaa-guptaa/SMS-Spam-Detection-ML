# SMS Spam Detection using Machine Learning

## Problem Statement
Spam messages are a common issue that affects users by spreading scams and unwanted content.

## Solution
This project uses a machine learning model to classify SMS messages as spam or non-spam
using text-based feature extraction.

## Tech Stack
- Python
- Scikit-learn
- TF-IDF Vectorization
- Naive Bayes Classifier

## Features
- SMS spam classification
- Lightweight ML pipeline
- Command-line interface

## How to Run
pip install -r requirements.txt  
python spam_classifier.py
